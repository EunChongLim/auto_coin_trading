# 대체 전략 아이디어

## 현재 상황

- 모델: LightGBM (3분 예측)
- 승률: 71.4%
- 수익률: +0.31% (낮음)
- 문제: 수익률이 낮음

## 대체 전략 제안

### 1. **가격 액션 기반 전략**

- 차트 패턴 인식 (양봉/음봉, 캔들 패턴)
- 지지/저항 레벨
- 추세선 분석
- **장점**: 간단하고 직관적
- **단점**: 주관적 판단

### 2. **변동성 기반 전략**

- ATR (Average True Range) 기반
- 볼린저 밴드 확장/수축
- 변동성 돌파 전략
- **장점**: 시장 변동성 활용
- **단점**: 횡보장에서 낮은 수익

### 3. **거래량 기반 전략**

- 거래량 급증 시 매수
- 거래량 급감 시 매도
- 거래량 가격 분석 (VPA)
- **장점**: 거래량은 선행 지표
- **단점**: 노이즈가 많음

### 4. **시간 필터 전략**

- 특정 시간대 거래
- 24시간 변동성 패턴 분석
- 거래량 전환 시간대 파악
- **장점**: 시간대별 특성 활용
- **단점**: 시간대 의존

### 5. **리스트 제이션 전략**

- 주문장 분석
- 매수/매도 호가 밀도
- 스프레드 분석
- **장점**: 거래 깊이 활용
- **단점**: API 제약

### 6. **멀티 코인 전략**

- 비트코인 + 알트코인
- 상관관계 분석
- 핵/인덱스 거래
- **장점**: 분산 투자
- **단점**: 복잡도 증가

### 7. **심층학습 모델**

- LSTM (순환 신경망)
- Transformer (어텐션)
- CNN + LSTM
- **장점**: 복잡한 패턴 학습
- **단점**: 학습 시간, 해석 어려움

## 추천 전략

### 즉시 적용 가능

1. **볼린저 밴드 전략** - 이미 BB 지표 사용 중
2. **변동성 기반 매매** - ATR 추가
3. **거래량 필터** - 거래량 기준 추가

### 중장기 개발

1. **LSTM 모델** - 시계열 학습
2. **강화학습** - 최적 전략 학습
3. **멀티 코인** - 리스크 분산
